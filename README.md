# echo-maze-game
# Echo Maze 回声迷宫

#项目简介

Echo Maze 是一款基于 HTML、CSS、JavaScript 和 Canvas 开发的网页迷宫小游戏。玩家处于黑暗迷宫中，无法直接看清完整地图，需要通过释放“声波”短暂探测周围墙体，并在迷宫中寻找钥匙和出口。

#游戏玩法
使用 WASD 或方向键控制角色移动，按空格键释放声波，声波经过的区域会短暂显示迷宫墙体，找到钥匙后才能进入绿色出口，以更少步数通关为目标

# 技术栈
HTML  CSS JavaScript Canvas

#项目亮点

使用 Canvas 实现游戏画面绘制，实现迷宫地图、角色移动、碰撞检测 设计“声波探测”机制，增强游戏创意， 实现钥匙、出口、步数统计和通关判断， 可部署到 GitHub Pages 在线游玩

# 运行方式

下载项目后，双击 index.html 即可在浏览器中运行。
